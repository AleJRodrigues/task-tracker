Task Tracker CLI 📝

A CLI-based task management system built with Java. This project allows you to organize daily chores with automatic data persistence.
🚀 Features

    Task CRUD: Add, view, and delete tasks from your list.

    Data Persistence: All tasks are automatically saved to a tasks.json file.

    Status Management: Easily toggle tasks as completed or pending.

    View Filters: List all tasks, only completed ones, or only unfinished ones.

🛠️ Technologies & Libraries

    Java 17+.

    Jackson Databind: Used for serializing Java objects into JSON data and vice versa.

    Lombok: Used to reduce boilerplate code by automatically generating Getters and Setters.

📂 Project Structure

    Task.java: The model defining a task (name, description, status).

    TaskFileHandler.java: Handles reading and writing data to the JSON file.

    MenuTask.java: Manages the user interface and menu options in the terminal.

Task Tracker CLI 📝

Um sistema de gerenciamento de tarefas via terminal desenvolvido em Java. O projeto permite organizar afazeres diários com salvamento automático de dados.
🚀 Funcionalidades

    CRUD de Tarefas: Adicionar, visualizar e excluir tarefas da sua lista.

    Persistência de Dados: Todas as tarefas são salvas automaticamente em um arquivo tasks.json.

    Gerenciamento de Status: Marcar tarefas como concluídas ou pendentes de forma simples.

    Filtros de Visualização: Listar todas as tarefas, apenas as concluídas ou apenas as inacabadas.

🛠️ Tecnologias e Bibliotecas

    Java 17+.

    Jackson Databind: Utilizado para transformar objetos Java em dados JSON e vice-versa.

    Lombok: Utilizado para reduzir o código repetitivo, gerando automaticamente Getters e Setters.

📂 Estrutura do Projeto

    Task.java: Modelo que define o que é uma tarefa (nome, descrição, status).

    TaskFileHandler.java: Responsável por ler e escrever as informações no arquivo JSON.

    MenuTask.java: Gerencia a interface do usuário e as opções do menu no terminal.
